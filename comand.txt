Check Docker & Compose:
	> docker --version
	> docker compose version
--------------------------------------------------------------
Check running containers:
	Check running containers: 			 > docker ps
	Check all containers (even stopped): > docker ps -a
--------------------------------------------------------------
Build & Run Everything: From your srcs/ directory:
	normal build			> docker compose up --build
	Or in background: 		> docker compose up --build -d
	Check logs:       		> docker compose logs
	Specific service logs: 	> docker compose logs (container name)
----------------------------------------------------------------------

Test NGINX (HTTPS)
	> curl -k https://localhost
	> curl -k https://yourlogin.42.fr
	> openssl s_client -connect localhost:443

--------------------------------------------------------------

Test MariaDB
	> docker exec -it mariadb bash
	> mysql -u root -p
	> SHOW DATABASES;

	docker exec -it wordpress bash
